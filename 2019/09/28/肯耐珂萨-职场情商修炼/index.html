沟通漏斗

  1. 第一层： 心里想的
  2. 第二层： 说出来的
  3. 第三层：对象听到的
  4. 第四层： 对方理解的
  5. 第五层： 行为产生

跨部门沟通的常见现象

  1. 关闭频道
    现象：没有在接受信息

    原因：内心地处，与自己无关

    影响：不欢而散
  2. 争夺频道
    现象：打断

    原因：意见不统一

    影响：争论、争执、争吵
  3. 分叉频道
    现象：无语

    原因：理解不一致

    影响：效率低下

跨部门沟通中如何“调频”

  • 了解和表达对方的需求
  • 合理的提问
  • 给与适当的肯定

同理心的三个维度

  • 换位思考
  • 认同对方的情感
  • 主动找出对方所关心或顾虑

同理心的自我锻炼

  • 当对方交办一项工作时，从对方的角度来理解这个工作
  • 激励他人时，了解对方信息需求，找准“兴奋点”
  • 批评他人时，照顾对方的面子、自尊，不揭伤疤
  • 与他人合作时，先理解对方的观点，看问题的角度，再介绍自己的想法

人际关系

何为人际关系

  • 信任
  • 关心
  • 互相满意

职场中“高人际”的四个特点

  1. 从对方角度看问题，尽力满足对方需求
  2. 有相互满意的人际关系
  3. 贡献自己的所能
  4. 需要帮助时也会提出并乐于接受帮助

职场中如何发展和维系人际关系

  1. 会议上多介绍自己的工作
  2. 多支持他人的工作
  3. 多请人来支援你的工作
  4. 分享工作中所有的信息
  5. 参加公司的内外活动
  6. 保持同理心

现实判断

为何现实判断

评价自己感受到的体验和客观存在的事物之间的一致性

  • 客观的区分内心世界的思想（情感）和外部世界的能力（过程）
  • 看到客观存在着的显示，而不是我们希望的显示或是害怕的现实

现实判断常见的六大误区

  1. 灾难化
  2. 自利偏见
  3. 证实偏误
  4. 晕轮效应
  5. 刻板印象偏误
  6. 沉没成本悖论

如何正确现实判断

  • 放弃第一印象总是对的的观念
  • 避免非黑即白的思维
  • 听取过来人的意见
  • 区别事实、评价、推论

冲动

控制冲动

ABCDE法则， A诱发事件->B想法->C情绪行为, 质疑B，从而避免坏的情绪行为C，产生新的结果E

控制冲动“四步法”

  1. 意识到并接受自己的情绪
  2. 弄清情绪真正产生的原因
  3. 做出愤怒的反应前，停一下6s
  4. ABCDE法则

压力

  • 由于自己能动用的资源无法满足工作和自身要求时所感受到的焦虑状态
  • 压力的大小既取决于压力源的大小，又取决于承担压力的对象强弱程度

留个压力征兆

  • 低迷
  • 易愤怒
  • 焦虑不安
  • 思维中断
  • 借口多
  • 注意力不集中

压力曲线

适当的压力有助于提高绩效

预警“P点”做好自我管理

捕捉P点常见信号

  • 再也忍受不了了
  • 没有办法面对
  • 怎样才能摆脱
  • 难道别人就不能帮帮我吗

P点不宜做的行为

  • 凡事拖拉
  • 把压力发泄到其他人身上
  • 沉迷电脑电视
  • 酗酒
  • 睡眠过渡

P点应该做的行为

  1. 写上自己面对的所有问题
  2. 把相关压力归类
  3. 从小问题逐步解决

有效的解压行为

  1. 良好的自我肯定
  2. 压力下做最好的自己
  3. 有规律的运动

如何保持乐观

  1. 相信事件最终能够解决的
  2. 积极想象成功的画面
  3. 多留意好的一面
  4. 避免灾难化思考
